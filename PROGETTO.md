## 1. Cos'√® il progetto

### 4.9 Normalizzazione numeri WhatsApp in E.164 (feb 2026)

**Obiettivo:** salvare tutti i numeri di telefono in formato standard E.164 (`+39XXXXXXXXXX`) per compatibilit√† futura con le API WhatsApp Business.

**Funzione `normalizePhone(raw)` aggiunta in `auth.js`:**
```js

// "348 123 4567"      ‚Üí "+39348123456"
// "0348 123 4567"     ‚Üí "+39348123456"
// "0039 348 123 4567" ‚Üí "+39348123456"
```
